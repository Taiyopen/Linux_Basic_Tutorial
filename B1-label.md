### 1. Install

(1) 安裝pip 

```bash
sudo apt install python3-pip 
```

(2) 安裝pyqt-tool

``` bash
sudo apt install pyqt5-dev-tools
```

(3) 安裝labelImg

``` bash
sudo pip3 install labelImg  
labelImg
```
